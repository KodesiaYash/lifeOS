# Health & Fitness Domain (`src/domains/health/`)

Nutrition, exercise, sleep, vitals, and wellness tracking.

**Status**: Scaffold only — implementation in Phase 1.

## Planned Features
- Meal logging with NLP parsing
- Exercise tracking with duration, type, intensity
- Sleep logging and analysis
- Vitals recording (weight, blood pressure, etc.)
- Nutrition coach agent
- Daily/weekly health summaries
